# Readme

misskeyで普段使用しているカスタムCSSを機能ごと？にしたものです、misskeyのバージョンによっては正常に動作しない可能性があります。

## wide_write_area.css

ノートを書くところをデカくしてます

## emoji_gyugyu_style.css

カスタム絵文字をギュウギュウにして表示します、とあるアプリに寄せた表記です

## long_notice_widget_area.css

ウィジェットの通知を縦に伸ばすことでプロフィールに飛びやすくなりリアクション返しが楽になります

## long_custom_css_write_area.css

カスタムCSSを記載する箇所を縦に伸ばすことで、長いCSSを見やすくなるよ
